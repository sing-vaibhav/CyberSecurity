Learning Roadmap

Where to start — and where to go next

1

Build Your Foundation (1–2 months)

Learn how computers and networks work. Understand IP addresses, TCP/IP, DNS, HTTP, and the Linux command line. This is the bedrock everything else builds on.

THM: Pre-Security Path OverTheWire: Bandit NetworkChuck on YouTube
2

Learn Security Fundamentals (2–3 months)

Study core security concepts — CIA triad, firewalls, encryption, authentication, vulnerabilities, and how common attacks work. Google's free certificate covers this well.

THM: Jr Pentester Path Google Cyber Certificate Professor Messer
3

Pick a Specialisation (3–6 months)

Choose your path: Offensive (Red Team) — ethical hacking, penetration testing, CTFs. Or Defensive (Blue Team) — SOC analyst, incident response, threat hunting. Both are in high demand.

Red: Hack The Box Red: Web Security Academy Blue: LetsDefend Blue: CyberDefenders
4

Get Certified (ongoing)

Certifications prove your skills to employers. Start with CompTIA Security+ (entry level), then progress to CEH, OSCP, or cloud security certs depending on your path.

CompTIA Security+ OSCP (Advanced) CEH
5

Build a Portfolio & Get Hired

Document your CTF writeups on a blog or GitHub. Try bug bounty on HackerOne or Bugcrowd. Contribute to open source security tools. Your public work is your CV.

HackerOne Bug Bounty Bugcrowd GitHub Portfolio
